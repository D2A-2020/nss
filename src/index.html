Nimble Storage System

Una aplicacion web para la gestion de un deposito mayorista. Funciones aspiradas:

  1. Control de espacios disponibles, en primera instancia, sin entorno visual
  2. Registro de cada articulo por codigo, stock.
  3. Busqueda de articulos
  4. Usuarios con distintos permisos: Cliente, Peon, Admin, Master
  5. Potencial escalable de la app: posibilidad de disennar mapa de estanterias por parte del usuario, ABML.
  6. Gestion de pedidos de mercaderia, asociando clientes y a peon con pedidos de mercaderia. Al hacer el pedido la mercaderia deberia descontarse del stock. El admin no tendria que dar de baja articuos de forma manual, aunque la funcion exista para exepciones.
  7. Control de envios de pedidos. Cronograma semanal. Rutas. Demora estimada. Ubicacion de unidades. Chat chofer-admins
  8. Control de etapas del pedido: Encargado, Preparandose, Pronto, Enviado al Cliente
  9. Entorno para crear pedido por parte del cliente
